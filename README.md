# solosafari

SoloSafiri India — a travel safety app with route planning, SOS features, chat, and traveler matching.

## Deployment

This repository contains a static frontend app in `frontend/` and a TypeScript backend in `backend/`.

- Frontend deploys best as a static site on Netlify, Vercel, or GitHub Pages.
- Backend can be hosted on Railway, Render, or another free Node.js host.
